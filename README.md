# Digital Clock

A simple real-time digital clock built using **HTML, CSS, and JavaScript**.  
The clock displays the current time and updates every second with a clean, centered layout.

The user interface is styled using **pure CSS** with a focus on simplicity and readability.

### Styling Highlights
